configuração da altura e largura

o tamanho padrão do iframe é de 300x150

ele tem 300 de largura e 150 de altura

nós podemos modificar esse tamanho no cod html porem usando css é melhor

rolagem do iframe

quando o o comteudo do site n cabe na barra de rolagem podemos usar a tag "scolling" que permite que o conteudo fique do tamanho do iframe com o "auto" e nos da a opçaõ de tirar as barras de rolagens com o "no"

no caso do brave, ele faz isso automaticamente, porem é bom colocar pq tal qual alguns navegadores n são compativieis com o iframe, outro navegadores podem n ir automaticamente

    o scolling tem 3 variaçoes
  1. auto
  2. yes
  3. no

iframe com conteudo local

para colocarmos um iframe com uma pasta logal basta colocamos o nome da pasta no lugar do src

se quisermos centralizar o iframe, temos que configura-lo como block, pq ele é inline, e dps colocar o "margin auto"

ex

         <style>
            body{
                background-color: bisque;
            }
            iframe{
                height: 300px ;
                width: 420px ;
                display: block;
                margin: auto;
            }
             </style>
